# GymManager - TFG

## Descripción
Sistema integral de gestión de gimnasios desarrollado como Trabajo de Fin de Grado (TFG) para el ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM).

El proyecto consta de tres componentes principales:
1. **gym-backend**: API REST desarrollada con Spring Boot.
2. **gym-android**: Aplicación móvil para socios del gimnasio.
3. **gym-web**: Panel de administración web para gestores.
4. **database**: Scripts SQL para la base de datos.

## Estructura del Proyecto
- `/gym-android`: Proyecto Android (Java/Kotlin).
- `/gym-backend`: Backend Spring Boot (Java).
- `/gym-web`: Frontend Web.
- `/database`: Modelado y scripts de base de datos.
- `Memoria_Final_GymManager.pdf`: Documentación detallada del proyecto.

## Requisitos
- JDK 17+
- Android Studio
- Node.js (para la web)
- MySQL/PostgreSQL (según configuración)
